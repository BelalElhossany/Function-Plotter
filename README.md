# Function Plotter

## Packages used
* PySide 2 for GUI.
* Matplotlib for figures
* pytest for testing

# Getting it working

## Dependencies
* pip install PySide2
* pip install matplotlib
* pip install pytest

## Run GUI
* python plotter.py

## Test
* pytest test.py
